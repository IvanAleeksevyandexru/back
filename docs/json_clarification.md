  ...

    {

      "type": "InfoScr",
      "label": "<p>Выписка содержит данные о стаже, перечисленных взносах, пенсионных коэффициентах и накоплениях</p><p><a id='ref_up'>Узнать подробнее</a></p>",

### Так и как с помощью ссылок:
* в разделе clarifications компоненты указывается список имен модальных окон
* в разделе clarifications JSON услуги пишется описание модального окна с именем указанным в компоненте
* На один раздел JSON.clarification услуги может ссылаться несколько компонент
